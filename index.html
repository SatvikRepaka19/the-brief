Overview

The Brief

A personal daily macro market intelligence dashboard. Python fetches data from FRED and Yahoo Finance, computes analytics, and writes data.json. This page reads that file and renders interactive Plotly charts. Refresh after running python generate.py to update all data.

How to Read
Regime Score (0–100)
Combines 5 signals into a Fear/Greed gauge. Each component contributes 0–20 pts. Scores 0–25 = Extreme Fear (sell everything), 46–55 = Neutral, 76–100 = Extreme Greed (be cautious). Based on rolling percentile rank of each signal.
Composite Score
Per-sector ETF composite = average of Return Z-score and Volume Z-score, clipped to ±3. Positive = recent outperformance + above-average volume (accumulation). Negative = underperformance + below-average volume (distribution).
Flow Z-Score
Rolling z-score of an ETF's daily volume vs its own 1-year history. Positive = above-average trading volume (attention/accumulation). Negative = below-average (quiet/distribution). Clipped to ±3.
Breadth (RSP/SPY)
Equal-weight vs cap-weight ratio. Rising = broader market participation (healthy bull). Falling = narrow leadership (fragile, often large-cap driven). Indexed to 1.0 at start of data window.
Cyc / Def Ratio
Equal-weight cyclical (XLK, XLF, XLY, XLI, XLB) divided by defensive (XLP, XLU, XLV). Rising = risk-on, markets pricing growth. Falling = risk-off, markets rotating to safety. A leading indicator for macro regime.
OAS Spreads
Option-Adjusted Spread = yield premium over Treasuries. HY OAS >500bps = elevated stress. IG OAS >200bps = caution. Wider spreads = higher perceived default risk = risk-off signal across all assets.
Breakeven Inflation
10Y Treasury yield minus 10Y TIPS yield. Market's 10-year inflation expectation. >3% = markets fear inflation. <1.5% = deflation fears. Fed targets ~2%. Rising breakevens with rising yields = stagflation concern.
Rolling Alpha
126-day compounded return of factor ETF minus SPY over same period. Positive = factor outperformed market. Negative = underperformed. Helps identify which investment style is working in the current regime.
Copper / Gold Ratio
Rising = copper (industrial) outperforming gold (safe haven) = growth expectations improving. Falling = gold outperforming = defensive positioning, growth fears, or geopolitical stress. Classic macro regime indicator.
Glossary
TermDefinitionBullish / Bearish Signal
Macro/Micro DispersionCross-sector std dev / (cross-sector + within-sector vol). Rolling 252d percentile.>0.75 = macro-driven. <0.25 = stock-picking market.
DrawdownCurrent price vs rolling all-time high in window: (price/cummax − 1) × 100%Lower drawdown = stronger uptrend. Deep DD = downtrend / recovery phase.
10Y-2Y SpreadDifference between 10-year and 2-year Treasury yields in percentage pointsPositive = normal curve. Negative (inverted) = recession signal.
Real Yield10Y Treasury yield minus inflation expectations (TIPS yield)High real yields hurt stocks. Negative real yields = stimulative environment.
52W High ProximityHow close current price is to 52-week high as % of the low-to-high range (0–100)>70% = near highs (momentum). <30% = near lows (potential value or deterioration).
Z-ScoreStandard deviations from rolling mean: (x − μ) / σ, clipped to ±3+2 = unusually high. −2 = unusually low. Near 0 = average.
